Viewing locally
============

## Getting started
* Clone the Repo ```git clone```
* Install NODE ``` npm install```
* Open MAMP/WAMP
* Go to MyPhpAdmin```-> create new database```
* Import the database inside Database folder
* In the Terminal run ``` npm start```
* In the Browser go ``` localhost:4000```
